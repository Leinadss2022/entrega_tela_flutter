# 📱 Tela de Perfil em Flutter

Uma interface de perfil desenvolvida com **Flutter** utilizando widgets nativos do framework. O projeto demonstra a construção de uma tela moderna, limpa e organizada, seguindo os princípios do Material Design.

## ✨ Funcionalidades

* AppBar personalizada
* Avatar circular com ícone de usuário
* Exibição de nome e profissão
* Cards com estatísticas do perfil
* Layout centralizado e organizado
* Interface responsiva para diferentes tamanhos de tela

## 🚀 Como executar

### Clone o repositório

```bash
git clone https://github.com/Leinadss2022/entrega_tela_flutter.git
```

### Entre na pasta

```bash
cd entrega_tela_flutter
```

### Instale as dependências

```bash
flutter pub get
```

### Execute a aplicação

```bash
flutter run
```

## 📱 Interface

A tela é composta pelos seguintes elementos:

* Barra superior (AppBar)

  * Botão de voltar
  * Título "Perfil"
  * Ícone de menu

* Perfil

  * Avatar circular
  * Nome do usuário
  * Cargo ou profissão

* Estatísticas

  * Posts
  * Seguidores
  * Seguindo
